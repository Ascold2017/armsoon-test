# armsoon-test

## Описание
Реализовать с помощью VueJS веб приложение — форма управления торговыми
ордерами тестового счета биржи bitmex, состоящая из 4 компонентов
- список инструментов (валютных пар) биржи
- история котировок для валютной пары
- форма для создания нового ордера
- история ордеров

Списки валютных пар и история котировок обновляются в реальном времени через websocket
В силу ограничений CORS и ограничений браузера - REST запросы ретранслируются через Node.js сервер

### Запуск проекта

1. Launch server: ```cd server && node app```

2. Launch frontend: ```npm run serve```
